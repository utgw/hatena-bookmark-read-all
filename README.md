## できること
* はてなブックマークの「あとで読む」タグを付けたブックマークを抽出できます。
* ワンクリックで、「あとで読む」を消す(既読にする)ことができます。
* ツークリックで、全ての「あとで読む」を消すことができます。
* そろそろ読みましょう。
